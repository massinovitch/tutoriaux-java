Connexion Jdbc

L objectif de ce tutoriel est montrer comment acceder à une base de données avec l'api jdbc. le tuto implémente aussi la couche dao et la couche service (une classe dao ne doit pas appeler une autre classe dao, ce principe est bien utilistré dans cet exemple, c'est la classe UtilisateurService qui appelle RoleDao pour charger un Role dans Utilisateur). pour exécuter ce tutoriel : :

  • créer la base de données dans mysql via le script scriptSql/bddMysql.sql
  • Pour tester la couche dao, lancer les classes junit qui se trouve dans le package com.persistence.test
  • Pour tester la couche service, lancer les classes junit qui se trouve dans le package com.service.test
  • il se peut que vous ayez une erreur à l'import du projet. il suffit d'aller supprimer les jar de lib et les reimporter
